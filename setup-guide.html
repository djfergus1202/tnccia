Quick Setup Guide

Get your Chemistry Platform running in 5 minutes

1
Download and Extract Package

Download the GitHub deployment package and extract it:

# Extract the package
tar -xzf chemistry-platform-github-final.tar.gz
cd chemistry-platform-github
The package contains all files needed for deployment
2
Environment Configuration

Set up your environment variables:

# Copy environment template
cp .env.example .env

# Edit with your API keys
nano .env
Required API Keys:
  • DISCORD_BOT_TOKEN - From Discord Developer Portal
  • GEMINI_API_KEY - From Google AI Studio
  • SESSION_SECRET - Random 32+ character string
  • DATABASE_URL - PostgreSQL connection string
Never commit your .env file to GitHub - it's already in .gitignore
3
Install Dependencies

Install Python and Node.js dependencies:

# Python dependencies
pip install -r requirements-github.txt

# Node.js dependencies (for Discord bot)
npm install
System Requirements:
  • Python 3.11+
  • Node.js 18+
  • PostgreSQL 12+
  • 2GB RAM minimum
4
Database Setup

Configure your PostgreSQL database:

# Local PostgreSQL (example)
createdb chemistry_platform

# Update DATABASE_URL in .env
DATABASE_URL=postgresql://username:password@localhost:5432/chemistry_platform
For production, use managed database services like Heroku Postgres or Railway
5
Run Applications

Start the Flask web application and Discord bot:

Terminal 1: Flask Web App
gunicorn main:app --bind 0.0.0.0:5000
Terminal 2: Discord Bot
node enhanced_discord_bot.js
Your platform will be available at http://localhost:5000
6
Verify Setup

Test that everything is working:

Health Check
curl http://localhost:5000/health
Expected Response:
{
  "status": "healthy",
  "features": {
    "video_generation": true,
    "discord_commands": 53,
    "research_system": true,
    "achievement_system": true
  }
}
Test Discord Bot
  • Invite bot to Discord server
  • Test command: /create-simple-video topic:chemistry
  • Check web interface for results
Automated Setup

Use the included automation script for guided setup:

# Make script executable
chmod +x deploy.sh

# Run automated setup
./deploy.sh

# Choose option 3: "Setup local development"

The script will:

  • Check system requirements
  • Install dependencies automatically
  • Create .env file from template
  • Provide next steps guidance
Common Issues
Port Already in Use
# Kill process on port 5000
lsof -ti:5000 | xargs kill -9

# Or use different port
gunicorn main:app --bind 0.0.0.0:8000
Discord Bot Token Invalid
  • Verify token in Discord Developer Portal
  • Check token in .env file
  • Ensure bot has proper permissions
Database Connection Failed
  • Check PostgreSQL is running
  • Verify DATABASE_URL format
  • Test connection manually
